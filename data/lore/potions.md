# Flame Bound — Potions & Consumables List

This file contains the current consolidated potion list (costs in **gold (g)**).

## Potion Use Summary
- **Drink a potion:** 2 AP. If **Engaged**, drinking **provokes an opportunity attack**.
- **Feed a potion to an ally (adjacent, willing):** 2 AP. Does **not** provoke.
- **No stacking:** A creature may benefit from **only one potion effect at a time**. Drinking a new potion ends any existing potion effect.
- **Temp HP:** Does not stack (use the higher value).
- **Potion dice:** Any **natural 1** on potion dice is rerolled once. Rolls of 2+ are kept.
